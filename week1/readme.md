Setelah instalasi berhasil, kita mulai menulis dan menjalankan program C# pertama.

## 📚 Materi Dasar yang Dipelajari

Materi difokuskan pada pemahaman fundamental berikut:

- Variable  
- Input & Output  
- Array  
- Perulangan (for loop)  
- Percabangan (if-else)  
- Operator  

---

## 🚀 Struktur Program

Program utama (`Program.cs`) menjalankan beberapa class pembelajaran berikut:

```csharp
Console.WriteLine("topik ngoding 25-02-2026\n");

// CLASS VARIABLE INPUT OUTPUT
Console.WriteLine("CLASS VARIABLE INPUT OUTPUT");
input_output.InputOutput();

// CLASS VARIABLE ARRAY FOR LOOP
Console.WriteLine("\nCLASS VARIABLE ARRAY FOR LOOP");
array_forloop.ArrayForLoop();

// CLASS VARIABLE IF ELSE
Console.WriteLine("\nCLASS VARIABLE IF ELSE");
if_else.IfElse();

// CLASS VARIABLE OPERATOR
Console.WriteLine("\nCLASS VARIABLE OPERATOR");
variable_operator.VariableOperator();
```

---

## 📂 Daftar Class

### 1️⃣ `input_output.cs`
Membahas:
- Deklarasi variabel  
- Input dari user (`Console.ReadLine()`)  
- Output ke console (`Console.WriteLine()`)  

### 2️⃣ `array_forloop.cs`
Membahas:
- Struktur data array  
- Perulangan menggunakan `for`  
- Mengakses elemen array  

### 3️⃣ `if_else.cs`
Membahas:
- Percabangan kondisi  
- Penggunaan `if`, `else if`, dan `else`  
- Logika pengambilan keputusan  

### 4️⃣ `variable_operator.cs`
Membahas:
- Operator aritmatika (`+`, `-`, `*`, `/`)  
- Operator perbandingan (`>`, `<`, `==`)  
- Operator logika
